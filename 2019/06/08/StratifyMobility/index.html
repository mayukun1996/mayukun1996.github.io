《当代中国社会分层与流动》

发表于 2019-06-08 | 分类于 读书笔记 | 阅读数
字数统计: 1.4k | 阅读时长 ≈ 4

前言

中国对于各个社会群体的评价经历了前后两个时期。第一个时期是从1949年到70年代末，全社会强调从政治角度评价各主要群体，阶级观点尤其受到推崇。第二个时期是1979年后，人们大多使用具有温和色彩的群体分类，更多地关注一些具体问题。

中国对社会分层与流动也存在一些总体性问题。首先是社会筛选机制不健全的问题。这里的筛选机制就是使社会的优秀分子能够更顺利地进入上层的机制。其次是社会分层体系紊乱的问题，比如“脑体收入倒挂”的问题。还有，身份制对于社会流动的阻碍。这是指户籍制度、人事制度对人们的身份有比较强制性的规定，因此，各种身份群体之间的流动较为困难。此外，其他总体性方面的问题还有：社会中间阶层弱小、单位办社会、劳动力市场不健全等等1。

第一章 社会分层与流动的基本概念

第一节 社会分层

1. 社会不平等

指的是个人或群体之间的任何可以评价出其价值高低的社会差异。

2. 社会差别（Social Differentiation）

个人、群体间的任何差异……主要是指在社会文化中任何具有重要意义的差异。

注意：社会差别并不一定有高低上下之分，但社会不平等则必然存在高低上下。

3. 社会分层（Social Stratification）

是指建立在法律或规则（rule）和结构基础上的、已经制度化的、比较持久的社会不平等的体系。

社会差别最为广泛，当对它予以价值评价时出现不平等，而只有当社会不平等已经形成了结构或制度化了后，才会出现社会分层……因此，人们也常常用“结构不平等”（Structured Inequality）来代替社会分层。

无高下之分的社会差别，经过评价被赋予了高下之差，当高下之差制度化之后，便形成了分层。

4. 身份（Status）

身份有两种涵义：

  • 指在社会结构中的一种位置，并有一系列的权利、义务与这种位置相联系；
  • 指在有高低上下之分的社会阶级、等级、职业等分层体系中的位置。

经典定义：“身份是在特定社会结构模式中所占据的一个位置2。”（Ralph Linton）

对身份的解释大致有六种涵义：

  • 在社会上的位置；
  • 在hierarchy（等级制度）中的位置；
  • 一种继承来的社会地位；
  • 任何一种用客观的特点（如职业、收入等）测量的地位；
  • 声望；
  • 权利与义务的集合。

分为两类：

  • 先赋身份（Ascribed Status），个人由生理原因先天获得的；
  • 成就身份（Achieved Status），个人经由自身的选择、活动、努力获得的身份。

第二节 社会流动

一、社会流动的涵义

社会流动指人们的地位、位置的变化……它是指个人或群体在社会分层结构与地理空间结构中位置的变化。社会学更关注人们在社会分层结构中位置的变化。

近现代的中国，由于政治、经济变迁剧烈，因而社会流动比率也依然较高。每一次大的革命运动和政治运动，都使一些社会上层集团流入下层，和使社会下层集团流入上层。过高的流动频率也造成了不少社会问题。

二、社会流动的类型

1. 按流动方向：垂直流动与水平流动

垂直流动是在社会分层体系中，位置的上、下移动。

根据垂直流动的状况，可以将社会区分为两类：

  • 开放的分层体系。存在较多垂直流动的社会；
  • 封闭的分层体系。社会流动受到严格限制的社会。

水平流动是在同一分层水平位置上的横向变化。

2. 按流动的主体：个人流动与群体流动

社会学更关注群体流动。

在一个封闭的分层体系中，社会流动主要表现为群体的流动。

3. 按结构性因素：结构性流动与非结构性流动

结构性流动是因社会结构的变迁而造成的人们地位的变化。

4. 按代际变迁：代际流动与代内流动

代际流动指父母与子女两代人之间的社会地位的变化。

5. 按流动速率：渐进的流动与骤变的流动

三、社会流动的途径

第一，职业的改变。

第二，经济上的成败。

第三，政治上的成败。

第四，教育成就。

第五，权力的控制。

第六，婚姻途径。
