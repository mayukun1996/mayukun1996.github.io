前言

时代的变化与技术的发展，已经使得原来很难进行的研究方法变得逐渐容易，尤其是数据时代的到来使得社会科学的研究发生了革命性的转变。但目前来说，我们仍然处在数字时代的早期阶段。写于这一阶段的本书主要面向两个群体：社会科学家和数据科学家。

这两个群体在这个时代分别有两个典型误解。对于数据科学家而言，有两个常见误解需要澄清。第一是认为数据越多越好。第二个是认为社会科学只是一种玩弄文字的游戏。而对于社会科学家而言，同样有两个误解。第一个是因为一些少数不真实的数据而彻底否定使用数字时代的工具进行研究；第二个是容易将现在和未来混淆，也就是“这类研究做得怎么样”和“这类研究将来会做得怎么样”。这本书更关心第二个问题，相对来说，对于计算社会科学的发展更乐观。这也具备了一定的科学社会学意义，任何新兴研究在刚开始的时候总是会受到来自社会各界，尤其是学术界的质疑。但通常来说，质疑往往代表旧有评判标准对新生事物的看法，其中一些具有建设性的质疑是可以纳入思考范围内的。

第一章：简介

一处墨迹

计算社会学的时代，收集到的数据在组织形式上与传统方法是相似的，但不同之处在于，不论是数据量还是数据搜集速度都大大提高了。

欢迎来到数字时代

数字时代的核心特征是：计算机随处可见。从计算机的发展史来说，它是作为拓展人类计算速度的工具而被发明出来的。几十年来，软硬件的发展使得它逐渐成为了人类社会中不可缺少的一部分，我们可以像它刚刚出现的时候那样，把它视为工具，但更加合适的看法则是，它已经以各种形式存在于社会的每个角落，仅仅赋予它工具的意义已经不能涵盖它的全部影响，它已经成为了人类本身的延伸。

在万物互联的时代，它扮演的角色将更加重要，我们不仅需要回答“它是作为谁的工具而存在”，还要回答“它的存在到底带来了哪些可能”。

研究设计

研究设计是一种将问题和答案连起来的“结缔组织”。这本书用接下来的四章内容——观察行为、提问、开展实验以及与他人合作——来说明研究设计的四种方法。

本书的主题

  1. 将现成物品和定制物品结合起来。所谓现成物品，是通过现成材料制作而成的物品。它在这里可以是现成的数据。所谓定制物品，是人们重新创造出来的新物品。它在这里可以是重新获取的数据。将这二者融合起来常常能得到耳目一新的结论。
  2. 道德伦理。在数字时代，研究者能得到有趣的结论，但也能以前所未有的方式伤害被研究者，因此道德伦理也值得讨论。

本书梗概

这是一个需要互相合作的时代，是一个跨学科交融的时代，自然科学、工程领域与社会科学将出现前所未有的合作，以往的学科划分可能不再适合今日的时代，但乐观的同时，我们也需要保持审慎的态度对待伦理道德面临的新挑战。不论如何，发展的问题只能通过发展来解决，停滞不前则永远不能得到有益于人类社会的结果。

第二章：观察行为

简介

模拟时代与数字时代是一对从信号理论当中借用过来的概念，它们的源头分别是“模拟信号”和“数字信号”。在信号理论当中，模拟信号是比较传统的信号传递形式，它不易远距离传输、容易失真，而数字信号则具有离散性，能够远距离传输、精度高。

在数字时代，人们的每一个行为，不论他或她是否意识到，都会在数字设备上产生一条行为痕迹，这些痕迹的有组织集合就构成了大数据。不仅企业拥有大数据，政府同样拥有。但不要被大数据的名头唬到，大数据也是社会研究的观察数据的一部分。粗略地看，观察数据指的就是“在不以某种方式进行干扰的情况下，通过观察某一社会系统所得来的任何数据”，也就是说，它在获得过程中既不与人交谈，也不改变人所处的环境。
